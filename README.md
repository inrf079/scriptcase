## What Is Scriptcase.host?

Scriptcase.host is a specialized hosting company that offers two types of hosting environments:

**Apps hosting** — for deploying finished Scriptcase-built applications (does not run Scriptcase itself)

**Developer hosting** — for running Scriptcase in the cloud so multiple developers can collaborate on projects without being tied to one machine

Every plan comes with cPanel, daily backups, free SSL, unlimited bandwidth, free site migration, PHP support, and database support for MariaDB/MySQL, PostgreSQL, Firebird, and SQLite. It's basically a Scriptcase-optimized shared hosting environment with some genuinely useful extras baked in.

One important clarification: **the plans do not include a Scriptcase license**. You'll need to purchase that separately through Scriptcase's official store if you don't already have one.

## Why Bother with Scriptcase Cloud Hosting?

Honestly, the case for it is pretty obvious if you've ever tried to manage a Scriptcase license across different machines. The standard desktop install is tied to one computer — which is fine when you only ever work at one desk, but starts feeling limiting fast.

With a cloud-hosted Scriptcase environment, your whole team can access the IDE from a browser, anywhere. No reinstalling, no moving licenses, no "but the dev environment is on Mike's laptop." The setup is already there, already optimized, already running.

Scriptcase.host also handles the infrastructure specifics that can trip people up: PHP version selection, module configuration, database compatibility (PostgreSQL, Firebird, SQLite alongside the usual MySQL). They claim their environment is specifically tuned for Scriptcase performance, and they offer a live demo environment if you want to test before committing.

## Who Should Consider Scriptcase.host?

If you're already a Scriptcase user — or planning to become one — and you want your development environment in the cloud, Scriptcase.host is probably the most straightforward option. The plans are priced reasonably, the infrastructure is already set up for Scriptcase specifically, and the support team understands the product.

If you're just looking for generic PHP hosting to deploy a finished application (not actively developing in Scriptcase), the Small and Enterprise apps plans at $3.95 and $9.13/mo are competitive with general shared hosting at that price point, especially given the database flexibility.

Where it's less compelling: if you need a lot of storage for a large project, the entry-level plans are fairly tight. The Plus developer plan's 20 GB is more reasonable for serious work.
